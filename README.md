# GTE Landing Page

A premium animated marketing landing page featuring glassmorphism design. Fluid animations and modern aesthetics that captivate visitors and drive conversions.

![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=next.js&logoColor=white)
![Framer Motion](https://img.shields.io/badge/Framer_Motion-0055FF?style=for-the-badge&logo=framer&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)

## Features

- **Glassmorphism UI with Layered Effects** — Frosted glass aesthetics with depth and transparency for a premium feel
- **Fluid Framer Motion Animations** — Smooth, physics-based animations that respond to user interaction
- **Fully Responsive Premium Design** — Pixel-perfect across all screen sizes from mobile to ultrawide
- **Performance Optimized** — Fast load times with optimized assets and efficient rendering

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| Next.js | React framework with optimized production builds |
| Framer Motion | Declarative animations and gesture handling |
| Tailwind CSS | Utility-first CSS for rapid, consistent styling |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/andrewpaik/gte-landing-page.git
cd gte-landing-page

# Install dependencies
npm install

# Run the development server
npm run dev
```

## Design Highlights

- **Hero Section** — Animated gradient background with floating glassmorphic elements
- **Feature Cards** — Hover-activated 3D transforms with glassmorphism effects
- **Scroll Animations** — Elements animate into view as users scroll through the page
- **CTA Sections** — Eye-catching call-to-action areas with animated gradients
